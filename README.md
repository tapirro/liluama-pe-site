# Liluama Perú

> Suplemento para la salud urinaria con manayupa, chanca piedra y plantas medicinales peruanas.

**[Visitar liluama.pe](https://liluama.pe)**

## Sobre este proyecto

Este repositorio contiene el sitio web de [Liluama Perú](https://liluama.pe), un recurso de salud basado en evidencia científica enfocado en **Salud Urinaria**.

### Ingredientes clave

Manayupa (*Desmodium molliculum*), chanca piedra (*Phyllanthus niruri*), cola de caballo, D-manosa

## Tecnología

- HTML5 + CSS3 (sitio estático)
- Hospedado en GitHub Pages
- Sin dependencias de JavaScript frameworks
- Optimizado para Core Web Vitals

## Evidencia científica

Todo el contenido está respaldado por:
- Estudios clínicos publicados en PubMed
- Monografías de la EMA (European Medicines Agency)
- Guías de la OMS sobre plantas medicinales
- Base de datos TRAMIL de plantas del Caribe y Latinoamérica
